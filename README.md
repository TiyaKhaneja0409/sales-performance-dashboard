# Sales Performance Dashboard

An interactive Power BI dashboard analyzing retail sales performance across regions, categories, and time.

## Overview
This dashboard visualizes key sales metrics to help identify performance trends and areas for business focus.

## Key Metrics (KPIs)
- Total Sales: 1.10M
- Profit Margin: 12%
- Total Profit: 132.52K
- Total Orders: 5K
- Average Order Value (AOV): 219.58

## Visuals
- **Total Sales by Year** — line chart showing sales growth from 2014–2017
- **Total Sales by Region** — bar chart comparing West, East, Central, and South
- **Total Sales by Category** — donut chart comparing Technology, Furniture, and Office Supplies

## Tools Used
Power BI Desktop, Power BI data modeling, interactive slicers/filters
